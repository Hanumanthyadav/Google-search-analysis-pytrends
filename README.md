# Google-search-analysis-pytrends
Python-based Google Trends analysis project using PyTrends to extract and visualize keyword search data. It identifies top countries by interest, plots global search distribution on a world map, analyzes time-wise trends, and compares multiple keywords. Useful for market research, SEO insights, and trend forecasting.
